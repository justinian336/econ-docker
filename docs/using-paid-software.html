Section 7 Using Paid Software

The examples I have used so far contain only open source code and data. However, many researchers use paid software with closed licenses, such as Stata, Matlab and ArcGIS.

Technically you can use these software distributions with Docker, but you need to have a license and make sure that it allows you to make use of said software in the way you intend. Here I provide some examples of popular paid packages and their usage with Docker.

7.1 Matlab

There is an official mathworks/matlab repository on Docker Hub. Its documentation contains all the information necessary to run Matlab within Docker.

7.2 Stata

As far as I know, there’s no official documentation regarding how to run Stata on Docker. However, you can find docker images on the dataeditors. Check also the GitHub repository at https://github.com/AEADataEditor/docker-stata.

7.3 ArcGIS Notebook Server

It is possible to perform analysis of geospatial data on ArcGIS Notebook Server over Docker. Please follow the instructions in the official website.
